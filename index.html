WebPerformance. Ejemplos. Sobre Nosotros.

¿Qué es?

Es la disciplina que se encarga de analizar un sitio web para recomendar cambios que ayuden a optimizar la velocidad de carga de la misma. La velocidad es un elemento crítico para ejecutar un sitio web exitoso y siempre debe ser una prioridad para los administradores de sitios web. Los sitios web de carga más rápida se benefician de un mejor compromiso del usuario. La velocidad de carga de una Web es un factor a tener en cuenta para el posicionamiento en buscadores.

¿Quienes somos?

Estudiantes de Diseño y Desarrollo Web del CETAV, por medio de este proyecto queremos dar a conocer las pricipales practicas que se llevan a cabo para lograr un buen web Performance para mejorar la experiencia de usuario, esto mediante elaboracion de un micrositio informativo .
